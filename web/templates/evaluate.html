4. Advanced Uncertainty Analysis

  

    

      
Three advanced analyses extending the standard evaluation: conformal prediction (coverage guarantees), aleatoric vs. epistemic uncertainty decomposition, and calibration degradation under distribution shift.

    


    
    
Conformal Prediction

    

      
Split conformal prediction uses the validation set to set a threshold guaranteeing that true labels fall in the prediction set with probability ≥ 1−α. Smaller average set size = more efficient (confident) predictions.

Aleatoric vs. Epistemic Uncertainty

    

      
Epistemic (model uncertainty, reducible) ≈ Mutual Information. Aleatoric (data noise, irreducible) = Predictive Entropy − MI. A high aleatoric ratio means the patches themselves are ambiguous, not that the model needs more training.
